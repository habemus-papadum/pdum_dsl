# design/archive — historical motivation & explorations

Pre-redesign material kept for the record. **Nothing here is an input to the
current implementation** — the active documents are `docs/desiderata.md`,
`design/proposed-architecture.md`, `design/implementation-plan.md`, and the
research corpus in `design/research/`.

| Item | What it was |
|---|---|
| `Generative Julia.ipynb` | The original motivating notebook (functional shader composition in Julia) that inspired the project |
| `demo_sketch.py` | The earliest API sketch, pre-M0 |
| `marimo/` | An exploratory side-investigation of marimo/anywidget notebook display (relevant again only when the notebook-canvas item on the far roadmap is pulled forward) |

Note the two *analysis* notes remain in `design/` proper — they are not
archived because they are still normative inputs: `dsl_caching_layer.md` is
the hazard checklist the marshaling/caching layers must satisfy (V4 and the
architecture cite it), and `closure_specialization.md` is the numba evidence
behind the caching thesis.
